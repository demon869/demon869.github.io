1.互联网为什么使用域名？

在互联网中，使用IP地址作为机器的绝对地址根本行不通，原因如下：

  • 不好记忆，例如告诉你一个某个网站的IP地址是“xxx.xxx.xxx.xxx”，可能根本记不住
  • 计算机可能常常更换IP地址，所以，通过IP地址去访问某台机器就会发生问题

DNS协议应运而生，主要解决把一个域名（比较好记忆的一串字符）和一个IP地址（比较难记忆）的关系对应起来

2.什么是DNS （Domain Name System）
